# DinkRound

A simple, fast web app for generating fair and balanced pickleball round-robin schedules. Perfect for organizing pickleball meetups, tournaments, and open play sessions.

**Live at:** [dinkround.com](https://dinkround.com)

## Features

- **Smart Scheduling**: Generates 8-round schedules that minimize partner repetition and distribute byes fairly
- **Flexible Configuration**: Supports 4-32 players across 1-8 courts
- **Progress Tracking**: Mark rounds as completed to track your event progress
- **Persistent Storage**: Automatically saves your schedule and completion status locally
- **Mobile Friendly**: Responsive design works great on phones and tablets
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks, no build step, no server required

## How to Use

1. Visit [dinkround.com](https://dinkround.com)
2. Enter the number of players (4-32)
3. Enter the number of courts available (1-8)
4. Click **Generate** to create your schedule
5. Mark rounds as complete by clicking the circle buttons in the header
6. Click **New Matchups** to generate a fresh schedule anytime

Your schedule automatically saves to your browser, so you can close the tab and return later without losing your progress.

## Origin

DinkRound is an extraction of the Round Robin scheduling block from the [Pickleball Ratings](https://github.com/ironprogrammer/pickleball-ratings) WordPress plugin. It was created to provide a standalone, easy-to-use scheduling tool that anyone can access without needing WordPress.
